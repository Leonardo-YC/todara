# Componentes de Todos (Core)

Conjunto de componentes visuales para la gestión de tareas.

## Estructura
- **TodoForm**: Formulario de creación con validación.
- **TodoList**: Lista filtrable de tareas.
- **TodoItem**: Item individual con edición inline y borrado.
- **TodoFilters**: Pestañas para filtrar (Todos/Activos/Completados).
- **TodoFooter**: Contador de tareas pendientes y limpieza.

## Tecnologías
- CSS Modules para estilos (sin Tailwind en JSX).
- Accesibilidad WCAG 2.1 (ARIA labels, navegación por teclado).
- Tipado estricto con TypeScript.
